# Products Server (Hapi + Prisma)

## Useful Commands

Install dependencies
```bash
npm install
```

Start backend in development mode (hot reloading, runs on port 5001)
```bash
npm run dev
```

Build TypeScript server
```bash
npm run build
```

Start compiled backend server
```bash
npm start
```

Generate Prisma Client
```bash
npm run prisma:generate
```

Create and apply database migrations
```bash
npm run prisma:migrate
```

Open Prisma Studio (visual database viewer)
```bash
npx prisma studio
```
